# Restaurant Management
This project is made for a restaurant's management system and implements the following:
```
-User Interface.
-Composite Design Pattern.
-Design by Contract.
-Observer Design Pattern.
-Data Serialization.
-FileWriter Class.
-Layered Architecture.
```
